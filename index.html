Лабораторна робота №1

Основи побудови та публікації HTML-документів

  1. Мета роботи
  2. Вивчити принципи побудови HTML-документа, його структуру та форматування. Розглянути можливості форматування тексту, організації списків, використання гіперпосилань, форм та графіки у web-документах.
    Познайомитись з особливостями табличного подання даних та недоліками використання невидимих таблиць для формування структури сторінки.
  3. Ознайомитись із системою контролю версій (CКВ) Git та можливостями сервісу Github Pages.
  4. Необхідні засоби

    Для виконання лабораторної роботи рекомендується використовувати таке відкрите програмне забезпечення:

  5. редактор коду MC Visual Studio Code або будь-який інший, який знайомий користувачу і забезпечує підсвічування синтаксису (Notepad++, Sublime Text, Atom, Brackets и т.п.);
  6. браузер Mozilla Firefox або Google Chrome;
  7. СКВ GIT (для роботи з сервісом GitHub Pages потрібен доступ до мережі Інтернет).
